# 📋 Task Dashboard

## Quick Reference

## 📥 All Open Tasks


```tasks
not done
sort by due
group by filename
```


## ⚠️ Overdue

```tasks
not done
due before today
sort by due
```

## 📌 Due Today

```tasks
not done
due on today
sort by priority
```

## 📅 Next 7 Days

```tasks
not done
due after today
due before in 7 days
sort by due
```

## ⏫ High Priority

```tasks
not done
priority is high
sort by due
```


## ✅ Recently Completed

```tasks
done
done after 7 days ago
sort by done reverse
limit 20
```

### Emoji meanings

| Emoji | Meaning                               | How to use                   | Example         |
| ----- | ------------------------------------- | ---------------------------- | --------------- |
| `📅`  | Due date                              | Add date after emoji         | `📅 2026-06-20` |
| `🏎️` | Start date (don't show before this)   | Add date after emoji         | `🛫 2026-06-18` |
| `⏳`   | Scheduled date (when to work on it)   | Add date after emoji         | `⏳ 2026-06-19`  |
| `✅`   | Done date (auto-added when completed) | Add date after emoji         | `✅ 2026-06-17`  |
| `⏫`   | High priority                         | Just the emoji, no value     | `⏫`             |
| `⏭️`  | Medium priority                       | Just the emoji, no value     | `⏩`             |
| `⏬`   | Low priority                          | Just the emoji, no value     | `⏬`             |
| `🔁`  | Recurring                             | Natural language after emoji | `🔁 every week` |

### Examples

```
- [ ] Simple task with no metadata
- [ ] High priority task with due date ⏫ 📅 2026-06-20
- [ ] Low priority task ⏬ 📅 2026-07-01
- [ ] Weekly standup notes 🔁 every Monday 📅 2026-06-23
- [ ] Start after vacation 🏎️ 2026-07-01 📅 2026-07-15
- [x] Already done task ✅ 2026-06-17
```
