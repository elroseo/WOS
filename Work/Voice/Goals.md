# Goals

## Current priorities (Q3 2026)

### 1. Ramp up as a CRE
- Complete onboarding training and Seismic assignments
- Pass GitHub Foundations certification (by mid-July)
- Build relationships with team and cross-functional partners
- Understand CRE tooling ecosystem (Dashboard, KQL, Splunk, LLM-Assist)

### 2. Build strong technical foundations
- GHES, GHEC, GHAS product knowledge
- Git workflows and CI/CD patterns
- Customer-facing troubleshooting skills
- Learn to read and query logs effectively (KQL, Splunk)

### 3. Develop CRE pillar skills
- **Mentorship**: Learn from seniors, document what I learn
- **Technical**: Build hands-on product expertise
- **Collaboration**: Contribute to team knowledge, pair with teammates
- **Program**: Understand account health and customer success patterns
- **Innovation**: Explore AI tooling (Copilot CLI, MCP, agents) to improve CRE workflows
- **Customer**: Practice customer communication, empathy, and problem-solving

### 4. Establish good working habits
- Daily briefing routine (/today skill)
- End-of-day summary for visibility
- Keep Obsidian vault organised and useful
- Regular 1-on-1 prep and follow-through

---

## How to use this file

The /today skill checks this file to ensure daily priorities align with these goals. If a day looks misaligned (all meetings, no learning; all admin, no customer work), it should flag that in the "Suggested Focus" section.

---

## Review cadence

- Weekly: Am I making progress on at least 2 of these?
- Monthly: Revisit and adjust priorities
- Quarterly: Full review with manager
