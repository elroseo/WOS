# People

Notes on people I work with regularly. These files help provide context for meetings, 1-on-1s, and collaboration.

## How to use

- One file per person
- Updated automatically by the /today skill (for upcoming meetings) or manually
- Use the template below to create new entries

## Template

When creating a new person file, include:
- **Role and team**
- **How I interact with them** (1-on-1, team meetings, project work)
- **What they're working on** (updated periodically)
- **Notes** (preferences, communication style, shared context)
