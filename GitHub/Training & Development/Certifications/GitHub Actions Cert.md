## Why this matters for CREs

Actions is one of the most common sources of enterprise customer questions. Understanding workflows, runners, secrets, and enterprise patterns is essential for troubleshooting and guiding customers.

## Study checklist

- [ ] Workflow anatomy: triggers, jobs, steps, runners
- [ ] GitHub-hosted vs self-hosted runners
- [ ] Secrets, variables, and environments
- [ ] GITHUB_TOKEN and workflow permissions
- [ ] Matrix strategy and concurrency
- [ ] Reusable workflows and composite actions
- [ ] Caching and artifacts
- [ ] OIDC for cloud provider auth
- [ ] Enterprise Actions policies and runner groups
- [ ] Billing and usage limits
