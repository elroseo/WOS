# Hub Notes

Captured content from GitHub's internal Hub for quick reference.

## How to use

1. Find something useful on The Hub
2. Create a new note in this folder (or ask Copilot to do it)
3. Use the `hub-note` template (or just paste the content)
4. Tag it with `#hub` and any relevant topics

## Searching

To find Hub content you've saved, use:

```tasks
tag includes #hub
```

Or just ask Copilot: *"What Hub notes do I have about X?"*

## Folders

Organize by topic as your collection grows:
- `hub-notes/onboarding/`
- `hub-notes/processes/`
- `hub-notes/tools/`
- etc.
