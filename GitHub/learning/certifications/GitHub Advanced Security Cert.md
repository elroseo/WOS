## Why this matters for CREs

GHAS is a major product area for enterprise customers and a core CRE focus. Understanding code scanning, secret scanning, and Dependabot helps you guide customers through enablement, rollout strategies, and alert triage.

## Study checklist

- [ ] GHAS licensing and enablement (org and enterprise)
- [ ] Code scanning and CodeQL setup
- [ ] Custom CodeQL queries
- [ ] Secret scanning: supported patterns and custom patterns
- [ ] Push protection
- [ ] Dependabot: alerts, security updates, version updates
- [ ] Security overview and enterprise reporting
- [ ] GHAS rollout strategies for large orgs

## Pillar connections

| Topic | CRE Pillar | How it helps |
