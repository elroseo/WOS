## Goal

Understand the GitHub App platform deeply, how they work, how they're scoped, and how to diagnose issues customers have with them.

## Key concepts

- GitHub Apps vs OAuth Apps vs PATs, when to use each
- GitHub App permissions model (repository, organization, account)
- Installation tokens and JWT authentication
- Webhooks: delivery, retries, payloads, signatures
- GitHub App installation and authorization flows
- App suspension and revocation
- Marketplace apps

## Learning tasks

- [ ] Register a GitHub App, install it on a repo, and authenticate as it
- [ ] Set up a webhook receiver and inspect incoming payloads
- [ ] Understand how to scope a GitHub App to minimum necessary permissions
- [ ] Diagnose a broken webhook delivery using the delivery log
