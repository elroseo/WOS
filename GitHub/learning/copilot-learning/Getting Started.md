# 2026-06-17 - Getting Started with Copilot CLI

## What we did

1. **Learned about Copilot customization**, three key file types:
   - **Agents** (`.agent.md`), custom personas with specific expertise
   - **Skills** (`SKILL.md`), on-demand instructions + scripts
   - **Instructions** (`.instructions.md`), path-specific or global guidance

2. **Created `~/copilot-config` repo** with starter templates:
   - Agents, skills, and instructions directories
   - `setup.sh` to symlink into `~/.copilot`
   - Pushed to GitHub as a private repo

3. **Practiced the pull request workflow:**
   - Branch, commit, push, `gh pr create`, `gh pr merge`
   - Full cycle from the command line, no browser needed

4. **Set up this Obsidian vault** for documentation
