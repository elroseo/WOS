**How to use: 
- Open Codespace 
- Run /welcome

	- use `/quickstart` for a faster startup. Type `/status` anytime to refresh your dashboard.




**Tips
💡 **TIP 1:** If you don't see the chat panel, click the Copilot icon or press `Ctrl+Alt+I`.

💡 **TIP 2:** Previous chat sessions live under **Sessions** in the chat header.

💡 **TIP 3:** The editor can look noisy. Ignore it unless I open a report or result file for you.

💡 **TIP 4:** When I generate a Markdown report, right-click the file tab and choose **Open Preview** to render it cleanly.

💡 **TIP 5:** The terminal is your status board. You usually do not need to type there.

💡 **TIP 6:** If the layout feels crowded, hide the Explorer, drag the chat divider left, or toggle the terminal with `Ctrl+``.
